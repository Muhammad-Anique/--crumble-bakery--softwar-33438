# Crumble Bakery Software Requirements

Welcome to the Crumble Bakery Software project! This repository contains the software requirements and development artifacts for the Crumble Bakery management system.

This project aims to develop a comprehensive bakery management software solution for Crumble Bakery, streamlining operations and enhancing customer experience.

- Staff scheduling
